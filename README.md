# Bunny Hop Adventure

A cute browser-based game where you control a bunny that hops over carrot obstacles.

## Play Online

To make this game available online, follow these simple steps:

### Option 1: GitHub Pages (Easiest)

1. Create a GitHub account if you don't have one at [github.com](https://github.com)
2. Create a new repository on GitHub
3. Upload all files from this project to your repository
4. Go to repository Settings > Pages
5. Under "Source", select "main" branch and click Save
6. Your game will be available at: `https://[your-username].github.io/[repository-name]/`

### Option 2: Netlify Drop (No account required)

1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop your entire project folder
3. Netlify will provide you with a random URL where your game is hosted
4. You can claim this site later by creating an account

### Option 3: Vercel (For more features)

1. Create a Vercel account at [vercel.com](https://vercel.com)
2. Install Vercel CLI or use their web interface
3. Deploy your project by connecting to your GitHub repository or uploading files
4. Your game will be available at a Vercel-generated URL

## Game Controls

- Press SPACE or UP arrow to make the bunny hop
- Avoid the carrot obstacles
- Try to get the highest score!

## Files

- `index.html` - Main HTML structure
- `style.css` - Game styling and appearance
- `game.js` - Game logic and mechanics

## Customization

Feel free to modify the game by:
- Changing colors in style.css
- Adjusting game difficulty in game.js
- Adding new features like power-ups or different obstacles

Enjoy your Bunny Hop Adventure!
